SnowflakeLibrary Documentation

This library provides keywords to interact with Snowflake, a cloud-based data warehousing platform, allowing users to execute SQL queries, manage databases, tables, users, roles, warehouses, and more.

Installation

pip install snowflake-connector-python pandas jinja2

Keywords

Open Connection

Open Connection  ${user}  ${password}  ${account}  ${warehouse}  ${database}  ${schema}

Opens a connection to a Snowflake database.

  • ${user}: Snowflake username.
  • ${password}: Snowflake password.
  • ${account}: Snowflake account name.
  • ${warehouse}: Snowflake warehouse name.
  • ${database}: Snowflake database name.
  • ${schema}: Snowflake schema name.

Close Connection

Close Connection

Closes the current connection to Snowflake.

Test Connection

Test Connection

Tests the connection to Snowflake by retrieving the current version.

Execute Query

Execute Query  ${query}

Executes an SQL query in Snowflake.

  • ${query}: The SQL query to execute.

Execute Query And Fetch Results

Execute Query And Fetch Results  ${query}

Executes an SQL query in Snowflake and fetches the results.

  • ${query}: The SQL query to execute.

Execute Stored Procedure

Execute Stored Procedure  ${procedure_name}  [${arg1}]  [${arg2}]  [...]

Executes a stored procedure in Snowflake.

  • ${procedure_name}: The name of the stored procedure.
  • ${arg1}, ${arg2}, etc.: Optional arguments for the stored procedure.

Execute Query From File

Execute Query From File  ${file_path}

Executes an SQL query stored in a file in Snowflake.

  • ${file_path}: The path to the file containing the SQL query.

Load Data From CSV

Load Data From CSV  ${table_name}  ${file_path}

Loads data from a CSV file into a Snowflake table.

  • ${table_name}: The name of the Snowflake table.
  • ${file_path}: The path to the CSV file.

Load Data From Parquet

Load Data From Parquet  ${table_name}  ${file_path}

Loads data from a Parquet file into a Snowflake table.

  • ${table_name}: The name of the Snowflake table.
  • ${file_path}: The path to the Parquet file.

Unload Data To CSV

Unload Data To CSV  ${query}  ${file_path}

Unloads data from a Snowflake query into a CSV file.

  • ${query}: The SQL query to execute.
  • ${file_path}: The path to save the CSV file.

Unload Data To Parquet

Unload Data To Parquet  ${query}  ${file_path}

Unloads data from a Snowflake query into a Parquet file.

  • ${query}: The SQL query to execute.
  • ${file_path}: The path to save the Parquet file.

Get Table Schema

Get Table Schema  ${table_name}

Gets the schema of a Snowflake table.

  • ${table_name}: The name of the Snowflake table.

Create Table

Create Table  ${table_name}  ${schema}
